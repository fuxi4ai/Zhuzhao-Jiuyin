# 🎯 预测者准确率体系

> **创建时间**: 2026-05-19 · 九儿
> **核心原则**：每条预测必须标明"谁说的"，才能评估每个源的可靠性

---

## 数据库结构

### `predictor_accuracy` 表

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | INTEGER | 主键 |
| `predictor_name` | TEXT | 预测者：小鲍老师/九儿/天哥/... |
| `date` | TEXT | 预测日期 |
| `prediction_type` | TEXT | 情绪周期/涨跌方向/板块/个股 |
| `prediction` | TEXT | 具体预测内容（如"冰点"、"看多"） |
| `actual_result` | TEXT | 实际结果（如"涨 0.52%"） |
| `is_correct` | INTEGER | 1=正确, 0=错误, NULL=待验证 |
| `next_day_return` | REAL | 次日上证涨跌幅（%） |
| `notes` | TEXT | 备注 |
| `created_at` | TEXT | 创建时间 |
| `verified_at` | TEXT | 验证时间 |

## 准确率验证逻辑

### 情绪周期 → 次日涨跌方向

| 情绪阶段 | 预期方向 | 判断逻辑 |
|---------|---------|---------|
| 主升/复苏 | 涨 | 看多预测，次日涨=正确 |
| 退潮/调整 | 跌 | 看空预测，次日跌=正确 |
| 冰点 | 涨 | 冰点后反弹预期，次日涨=正确 |
| 过热 | 跌 | 过热后回调预期，次日跌=正确 |
| 震荡 | ±1% 以内 | 小幅波动=正确 |

### 板块/个股预测

板块/个股预测需要后续对接行情数据（tushare 个股日K）才能自动验证。

## 历史数据回填（2026-05-19 首次执行）

### 小鲍老师 · 总准确率：53.0%（35/66）

| 阶段 | 样本 | 正确 | 错误 | 准确率 | 次日均涨跌 |
|------|------|------|------|--------|-----------|
| 过热 | 4 | 4 | 0 | **100%** | +0.14% |
| 退潮 | 13 | 7 | 3 | **70.0%** | -0.09% |
| 调整 | 24 | 9 | 8 | 52.9% | -0.03% |
| 震荡 | 2 | 1 | 1 | 50.0% | -0.86% |
| 冰点 | 30 | 11 | 14 | 44.0% | -0.02% |
| 主升 | 3 | 1 | 2 | 33.3% | +0.08% |
| 复苏 | 5 | 2 | 3 | 40.0% | -0.09% |

### 关键发现

1. **总准确率 ≈ 抛硬币**（53%），小鲍老师的情绪周期判断整体参考性有限
2. **过热判断最准**（100%），但样本仅 4 次
3. **冰点判断最不准**（44%），30 次只对了 13 次
4. **退潮判断较可靠**（70%），13 次对了 9 次
5. 15 条待验证（预测日期无次日 K 线数据）

## 日常维护

### 每日自动验证
脚本：`db/verify_daily_predictions.py`
- 每日复盘入库后自动运行
- 用 tushare 次日数据验证前一天预测
- 更新 `predictor_accuracy.is_correct` 和 `verified_at`

### 手动查询准确率
```sql
-- 某预测者总准确率
SELECT predictor_name,
       COUNT(*) as total,
       SUM(CASE WHEN is_correct=1 THEN 1 ELSE 0 END) as correct,
       ROUND(100.0 * SUM(CASE WHEN is_correct=1 THEN 1 ELSE 0 END) / 
             SUM(CASE WHEN is_correct IS NOT NULL THEN 1 ELSE 0 END), 1) as accuracy_pct
FROM predictor_accuracy
WHERE is_correct IS NOT NULL
GROUP BY predictor_name;

-- 某预测者各阶段准确率
SELECT prediction, COUNT(*),
       ROUND(100.0 * SUM(CASE WHEN is_correct=1 THEN 1 ELSE 0 END) / COUNT(*), 1) as accuracy_pct
FROM predictor_accuracy
WHERE predictor_name = '小鲍老师' AND is_correct IS NOT NULL
GROUP BY prediction;
```
