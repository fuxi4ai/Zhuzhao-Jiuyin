## 核心设计理念

1. **四维度框架**: 外围定价 → 行业主线 → 情绪技术 → 交易策略，层层递进
2. **数据分离**: 每日复盘记录与维度分析数据分离，避免冗余
3. **可扩展性**: 预留数据源字段，支持多渠道数据导入
4. **量化情绪周期**: 6 项指标综合评分，定位情绪周期阶段
5. **双轨验证**: 量化结果与天哥直觉对比，持续优化判断
6. **标的追踪**: 重点标的独立表，记录每次关注的原因和方向

## 产品定位与架构

> 来源：PRD v2.0 + 金融工具化架构，整合于 2026-05-14

### 一句话定义

基于历史复盘数据 + 量化指标 + 行情回测的 **A 股情绪周期决策工具**

### 三层体系架构

```
┌─────────────────────────────────────────────┐
│              🎯 预测层 (Predictive)          │
│  情绪周期定位 → 策略推荐 → 每日预测报告       │
│  历史相似日匹配 → 胜率预测 → 风险预警         │
├─────────────────────────────────────────────┤
│              📊 回测层 (Backtesting)         │
│  方向性信号提取 → 行情数据对接 → 胜率/收益    │
│  按情绪阶段分组回测 → 板块回测 → 信号排名     │
├─────────────────────────────────────────────┤
│              📝 记录层 (Records) ✅ 已完成   │
│  四维度数据库 → 信号提取引擎 → 快速录入工具   │
│  板块标准化 → 个股提取 → 双轨情绪周期         │
└─────────────────────────────────────────────┘
```
