## 三、遗漏的高价值数据 ⚠️

### 3.1 重点标的/个股（第 5 列）

**价值**: ⭐⭐⭐⭐⭐ 最高！
**现状**: ❌ 完全未提取

**示例数据**:
```
看多：AI、减肥药
看多：AI-SORA、存储
看多：钨矿
看多：新凯来（半导体设备）
看多：光模块、存储、半导体设备
看多：AI 硬件、医药
看空：高位股
```

**分析**:
- 小鲍直接给出看好的板块和个股
- 这是**最有回测价值**的数据 — 可以直接验证"看多→次日涨跌"
- 包含板块（AI、光模块）和概念（减肥药、SORA）
- 部分记录有"看空"信号

### 3.2 消息面/事件驱动
