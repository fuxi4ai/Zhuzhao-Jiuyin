## 设计注记

1. **锚的三种用法**（后续接 closure_engine 的 `us_echo` 字段，本期只入库）：
   - 先行：美股锚 5/20 日对 SPY 超额转正 → A股对应主线未启动 = 跨市场 gap，open 信号加分
   - 印证：A股主线 closing 期间美股锚同步超额 = 全球共振，兑现更可信
   - 背离：A股 closing 但美股锚走弱 = 纯内资叙事，警示降权
2. **三个口径锚**（TSM/AAPL/LLY）只作行业温度计，不作同向印证——A股对应主线的逻辑分别是国产替代/华为链/BD出海，与美股标的非同向。
3. ALM 历史仅自 2025-07，钨信号（251xxx 批）勉强够；更早回测留空。
