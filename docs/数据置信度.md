# 数据置信度体系

> **最后更新**: 2026-05-11
> **说明**: P0/P1/P2/P3 为**置信度**标识（非优先级）
> 
> **核心原则**: 同一数据类型，不同平台的置信度相同。
> 例如：tushare pro 金融数据 = P0，gangtise 金融数据 = P0。
> 区别只在**优先级**——先走哪个渠道获取。

---

## 置信度定义（数据质量）

| 标识 | 含义 | 说明 |
|------|------|------|
| **P0** | 基准数据 | 最高置信度，可独立采信为基准 |
| **P1** | 高置信度 | 可靠信息源，可独立采信 |
| **P2** | 中等置信度 | 可参考，需交叉验证 |
| **P3** | 低置信度 | 仅参考，不采信为结论 |

---

## 数据置信度（按数据类型）

### P0 · 基准数据

| 数据类型 | 来源（任意平台） | 说明 |
|---------|----------------|------|
| 行情数据（日K/OHLCV） | tushare pro / gangtise / 句芒 | 交易所原始数据，不同平台置信度相同 |
| 估值分位（PE/PB） | tushare pro / gangtise | 同上 |
| 财务报表 | tushare pro / gangtise | 同上 |
| 指数数据 | tushare pro / gangtise | 同上 |

### P1 · 高置信度

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| 新闻内容（宏观/政策/深度） | 财新 | 包年无限流，专业记者采写 |
| 实时资讯/板块异动 | 冈底斯投研 | 实时新闻流 |
| 外资研报 | 哥哥提供的免费渠道 | 高可信度 |
| 专家纪要（含"帕米尔"/"专家"） | 冈底斯 gts_summary | 专家访谈纪要 |

### P2 · 中等置信度

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| 语义知识库（独家供需） | 冈底斯 gts_kb | 独家数据，需交叉验证 |
| 国内研报 | 冈底斯 gts_report | 国内机构研报 |
| 会议纪要（普通） | 冈底斯 gts_summary | 不含"帕米尔"/"专家" |

### P3 · 低置信度

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| 国外研报（冈底斯积分） | 冈底斯 | 需付积分，非优先 |

---

## 优先级（数据获取渠道选择）

> 同一数据类型有多个可用渠道时，按优先级顺序查询。
> **优先级不影响置信度**——不同平台获取的同一数据置信度相同。

### 行情类

| 优先级 | 渠道 | 置信度 | 说明 |
|--------|------|--------|------|
| **第 1 优先** | tushare pro | P0 | 年包 5000 积分，不限流量，已付费 |
| **第 2 优先** | 句芒行情库 | P0 | tushare 拉取的副本，覆盖 2 年历史 |
| **备用** | gangtise gts_quote | P0 | 需要消耗积分，仅在上述不可用时使用 |

**规则**: 日K行情、估值分位、财务报表等能从 tushare pro 获得的数据，统一走 tushare pro 渠道。

### 新闻类

| 优先级 | 渠道 | 置信度 | 说明 |
|--------|------|--------|------|
| **第 1 优先** | 冈底斯投研 | P1 | 实时资讯/板块异动，新闻场景优先 |
| **第 2 优先** | 财新 | P1 | 包年无限流，深度报道/宏观政策 |

**规则**: 新闻场景冈底斯优先于财新（两者置信度相同 P1）。

### 研报/纪要类

| 优先级 | 渠道 | 置信度 | 说明 |
|--------|------|--------|------|
| **第 1 优先** | 哥哥提供的外资研报（免费） | P1 | 免费渠道 |
| **第 2 优先** | 冈底斯 gts_summary（含"帕米尔"/"专家"） | P1 | 专家纪要 |
| **第 3 优先** | 冈底斯 gts_kb | P2 | 独家供需数据 |
| **第 4 优先** | 冈底斯 gts_report（国内研报） | P2 | 国内机构研报 |

---

## 冲突处理规则

| 场景 | 规则 |
|------|------|
| 行情数据冲突 | tushare pro 覆盖句芒（非置信度差异，而是约定基准源） |
| 新闻场景 | 冈底斯优先于财新（实时性更好） |
| 纪要置信度判断 | 名称含"帕米尔"或"专家"→ 提为 P1 |
| 研报来源 | 哥哥提供的（P1）> gts_report 国内研报（P2） |
