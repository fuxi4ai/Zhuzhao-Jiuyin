# 📊 HTML 报告模块设计

> **状态**: 规划中  
> **技术选型**: ECharts + Jinja2 (Python)  
> **报告类型**: 日报/周报/月报

---

## 一、报告类型与频率

| 类型 | 生成频率 | 数据范围 | 核心内容 | 推送时机 |
|------|---------|---------|---------|---------|
| **日报** | 每日 | 当日复盘 | 情绪阶段 + 信号 + 次日预测 | 每晚 22:00 |
| **周报** | 每周五 | 本周 5 个交易日 | 情绪走势 + 板块轮动 + 胜率 | 周五 23:00 |
| **月报** | 每月底 | 本月所有交易日 | 周期回顾 + 信号排名 + 策略总结 | 月末最后 1 天 |

## 三、数据源与 SQL 查询模板

### 日报查询
```sql
-- 当日复盘数据
SELECT * FROM recap_daily WHERE date = ?;
SELECT * FROM dim1_external_pricing WHERE date = ?;
SELECT * FROM dim2_sector_themes WHERE date = ?;
SELECT * FROM dim3_sentiment_tech WHERE date = ?;
SELECT * FROM dim4_trade_plan WHERE date = ?;

-- 相似历史日 (前 5)
SELECT signal_date, next_day_return, outcome 
FROM signal_backtest 
ORDER BY ABS(next_day_return - ?) LIMIT 5;
```

### 周报查询
```sql
-- 本周情绪走势
SELECT date, emotion_stage, total_score 
FROM cycle_quant 
WHERE date BETWEEN ? AND ? 
ORDER BY date;

-- 本周板块提及
SELECT sectors_bullish, COUNT(*) as mentions 
FROM dim2_sector_themes 
WHERE date BETWEEN ? AND ? 
GROUP BY sectors_bullish 
ORDER BY mentions DESC LIMIT 10;

-- 本周信号胜率
SELECT sig_type, COUNT(*), 
       SUM(CASE WHEN outcome='HIT' THEN 1 ELSE 0 END) as hits
FROM signal_backtest 
WHERE signal_date BETWEEN ? AND ? 
GROUP BY sig_type;
```

### 月报查询
```sql
-- 月度情绪分布
SELECT emotion_stage, COUNT(*) as days 
FROM cycle_quant 
WHERE strftime('%Y-%m', date) = ? 
GROUP BY emotion_stage;

-- 月度信号排名
SELECT signal_date, sig_type, next_5d_return, outcome 
FROM signal_backtest 
WHERE strftime('%Y-%m', signal_date) = ? 
ORDER BY next_5d_return DESC;

-- 月度累计收益
SELECT signal_date, 
       SUM(next_day_return) OVER (ORDER BY signal_date) as cumulative_return 
FROM signal_backtest 
WHERE strftime('%Y-%m', signal_date) = ?;
```

## 四、可视化方案详细设计

| 数据类型 | 图表类型 | ECharts 组件 | 用途 |
|---------|---------|-------------|------|
| 情绪周期趋势 | 面积图 | `line` (areaStyle) | 展示 5 日/月情绪演变 |
| 板块热度 | 词云 | `wordCloud` | 快速识别热门板块 |
| 信号胜率对比 | 柱状图 | `bar` | 对比不同信号效果 |
| 累计收益曲线 | 折线图 | `line` (stack) | 展示策略累计收益 |
| 涨价传导链 | 桑基图 | `sankey` | 展示产业链传导 |
| 情绪阶段分布 | 饼图 | `pie` | 展示阶段占比 |
| 涨跌停对比 | 双轴图 | `bar` + `line` | 涨停 vs 跌停 |
| 量化评分雷达 | 雷达图 | `radar` | 6 指标评分对比 |
| 关键事件日历 | 热力图 | `calendar` + `visualMap` | 消息催化时间分布 |
| 收益瀑布图 | 瀑布图 | `bar` (waterfall) | 策略收益贡献分解 |
